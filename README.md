# Shop Go

Небольшой магазин электроники на Go, Gin и SQLite.

## Запуск

```powershell
cd catalog
go run .
```

После запуска магазин будет доступен по адресу:

```text
http://localhost:8080
```

## Что входит

- витрина магазина;
- каталог и фильтры;
- карточки товаров с реальными изображениями;
- корзина, избранное и заказы;
- админ-панель для товаров, категорий, брендов и заказов;
- база `catalog/catalog.db` с текущими товарами.
